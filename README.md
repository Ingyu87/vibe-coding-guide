## GitHub에서 편집하는 법

### 1단계. 담당 파일 열기
위 배정표에서 내 파일 경로 확인 → 이 페이지에서 `03_원고/` 폴더 클릭 → 담당 파일 클릭

### 2단계. 편집 시작
파일 오른쪽 위 **연필 아이콘(✏️)** 클릭

### 3단계. 내용 작성
- 파일 맨 위 `<!-- 상태: 미시작 -->` → `<!-- 상태: 진행중 -->` 으로 바꾸기
- 내용 작성

### 4단계. 저장(커밋)
- 페이지 오른쪽 위 **Commit changes** 버튼 클릭
- 메모란에 간단히 작업 내용 입력 (예: `1장 초안 작성`)
- **Commit changes** 버튼 클릭

### 5단계. 완료 표시
작성 완전히 끝나면 파일 맨 위를 `<!-- 상태: 완료 -->` 로 변경 후 저장

## 꼭 지켜야 할 것

- **담당 파일만 수정하세요.** 다른 사람 파일은 건드리지 않습니다.
- **`00_제작기준/` 파일은 수정하지 마세요.** 집필 기준 문서입니다.
- 완료 표시는 완전히 끝났을 때만 합니다. 중간 저장은 `진행중` 상태로 두세요.
- 궁금한 점은 백인규에게 문의하세요.
