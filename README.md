## 지침 파일 다운로드 방법

### 전체 다운로드 (처음 한 번)
1. 이 페이지 오른쪽 위 **`<> Code`** 버튼 클릭
2. **`Download ZIP`** 클릭
3. 압축 해제 후 원하는 폴더에 저장

---

## 에이전트에게 줄 파일

다운로드한 폴더에서 아래 파일을 에이전트에게 제공하세요.

### 항상 줄 것 (모든 파트 공통)
| 파일 | 역할 |
|---|---|
| `00_제작기준/집필작업지침.md` | 문체·구조·출처 활용 기준 전체 |
| `00_제작기준/toc_detail.md` | 담당 장의 집필 상세 가이드 |

### 담당 파트 쓸 때 추가로 줄 것
| 파일 | 역할 |
|---|---|
| `00_제작기준/사전조사_출처_분석표.md` | 어떤 참고자료를 어느 장에 쓸지 |
| `02_출처_MD/` 안의 관련 파일 | 실제 참고자료 내용 |

> 참고자료 매핑은 `사전조사_출처_분석표.md`에서 담당 파트 행을 찾아 확인하세요.

## 에이전트 프롬프트 예시

에이전트 대화 시작 시 아래처럼 시작하면 됩니다.

```
참고자료로 [관련 출처 파일명]도 함께 줄게.
```
