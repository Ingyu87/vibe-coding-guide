## 집필 지침 파일 3개 — 역할 구분

| 파일 | 한 줄 역할 | 언제 쓰나 |
|---|---|---|
| `00_제작기준/CLAUDE.md` | 어떻게 쓸 것인가 (문체·규칙·톤) | Claude Code 자동 로드 |
| `00_제작기준/집필작업지침.md` | 왜·누구를 위해 쓸 것인가 (목적·출처 활용) | 모든 에이전트 공통 |
| `00_제작기준/toc_detail.md` | 무엇을 쓸 것인가 (장별 목적·내용·체크리스트) | 담당 장 섹션만 |

## 에이전트별 사용법

### Claude Code
1. 이 레포를 ZIP으로 다운로드 후 압축 해제
2. **해당 폴더에서 Claude Code 실행** → `CLAUDE.md` 자동 로드됨
3. 아래 프롬프트로 시작

```
집필작업지침.md 읽고,
toc_detail.md에서 [담당 파트] 섹션 확인해줘.
관련 출처는 사전조사_출처_분석표.md 보고 02_출처_MD/ 에서 찾아줘.
03_원고/[담당파일].md에 초안 작성해줘.
```

---

### Cursor
1. 이 레포를 ZIP으로 다운로드 후 Cursor로 폴더 열기
2. 채팅창에서 `@` 로 파일 첨부하여 시작

```
@집필작업지침.md @toc_detail.md
[담당 파트] 초안을 03_원고/[담당파일].md에 작성해줘.
참고자료는 @사전조사_출처_분석표.md 보고 02_출처_MD/ 에서 찾아줘.
```

---

### 안티그래비티
1. 이 레포를 ZIP으로 다운로드
2. 안티그래비티에 파일 업로드하거나 내용 붙여넣기

```
첨부한 집필작업지침.md와 toc_detail.md([담당 파트] 섹션)를 읽고
[담당 파트] 초안을 작성해줘.
참고자료는 사전조사_출처_분석표.md 보고 알려주면 해당 출처 파일도 줄게.
```

---

### Codex
1. 이 레포를 ZIP으로 다운로드
2. 집필작업지침.md 내용을 시스템 프롬프트 또는 첫 메시지에 붙여넣기

```
[집필작업지침.md 내용 붙여넣기]
위 지침에 따라 toc_detail.md의 [담당 파트] 섹션 기준으로
[담당 파트] 초안을 작성해줘.
```
